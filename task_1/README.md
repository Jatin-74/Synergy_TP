Task 1: GitHub, Virtual Environment, and Linux Basics


Description:


This folder contains the setup for Task 1 of the Synergy\_TP project.



It demonstrates how to set up a Python virtual environment, install



packages, and use basic Linux commands.



Setup Instructions:


1\. Clone the repository

git clone https://github.com/divyanshu1888/Synergy\\\_TP.git



cd Synergy\_TP



2\. Create and activate virtual environment


cd task\_1



python -m venv task\_1/venv



source task\_1/venv/Scripts/activate



3\. Install requirements

pip install -r task\_1/requirements.txt



4\. Run the Python script

python task\_1/src/hello.py
